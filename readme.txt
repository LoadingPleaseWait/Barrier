We are not using the default package. Instead, we are using the package "us.loadingpleasewait.barrier".
The fully qualified names of those two classes are now "us.loadingpleasewait.barrier.monitorBarrier" and "us.loadingpleasewait.barrier.semaphoreBarrier".

To compile, go to src folder and run the command "javac -d ../bin us/loadingpleasewait/barrier/*.java". This will make sure all your class files are generated in the bin folder.

To execute, go to bin folder and run the command "java us.loadingpleasewait.barrier.Test".
